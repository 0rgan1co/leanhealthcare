# Lean Healthcare

Landing del servicio de **modernización y mejora continua en salud** — pública y privada — con enfoque Lean Healthcare: trazabilidad integral, ciclos cortos medibles y modelo humano + IA.

## Stack

- `index.html` único, Tailwind vía CDN, Inter desde Google Fonts.
- Sin build, sin dependencias. Edición y deploy en minutos.

## Deploy

GitHub Pages desde `main`, root `/`.
URL: https://0rgan1co.github.io/leanhealthcare/

## Estructura de contenido

1. Hero y propuesta de valor
2. Tres miradas: paciente · profesional · procesos
3. Modelo zentauro (humano + IA)
4. Las 6 etapas de trazabilidad integral (A–F)
5. Matriz esfuerzo/impacto por etapa
6. 5 preguntas estratégicas para la dirección
7. CTA de contacto

## Edición rápida

- Texto y copy: todo vive en `index.html`.
- Correo del CTA: buscar `mailto:` y reemplazar.
- Paleta: ver bloque `tailwind.config` dentro de `<head>`.
